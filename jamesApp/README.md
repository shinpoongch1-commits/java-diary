# 야고보서 구간반복 (Java 데스크톱 버전)

원본 HTML(`야고보서_구간반복_v2.html`)을 자바 Swing 데스크톱 앱으로 이식한 버전입니다.
야고보서 1~5장(총 108절), 한글(개역개정) / 영어(NIV 1984) 본문이 모두 포함되어 있습니다.

## 폴더 구성
```
JamesApp/
 ├─ src/app/       자바 소스 코드
 │   ├─ Main.java          진입점
 │   ├─ MainFrame.java     화면(UI) + 재생 로직
 │   ├─ TtsService.java    OS별 음성 합성(TTS) 연동
 │   ├─ BibleData.java     야고보서 108절 본문 데이터
 │   └─ Verse.java         절 데이터 모델
 ├─ bin/            컴파일된 .class 파일 (이미 컴파일되어 있음)
 └─ JamesRepeat.jar 바로 실행 가능한 실행 jar
```

## 실행 방법

### 1) 미리 빌드된 jar로 바로 실행 (JDK 또는 JRE만 있으면 됨)
```
java -jar JamesRepeat.jar
```

### 2) 소스에서 직접 빌드하기 (JDK 11 이상 필요)
```
cd src
javac -d ../bin app/*.java
cd ../bin
java app.Main
```

## 음성(TTS) 재생 방식 — "원본과 동일한 방식"
브라우저의 `speechSynthesis`가 운영체제에 이미 설치된 음성을 사용하는 것과 동일하게,
이 자바 앱도 **별도 설치 없이 OS에 내장된 음성**을 그대로 호출합니다.

| OS | 사용 방식 | 참고 |
|---|---|---|
| **macOS** | 내장 `say` 명령어 | 한국어 "Yuna(유나)" 음성이 기본 내장되어 있어 바로 재생됩니다. |
| **Windows** | PowerShell + System.Speech(SAPI) | 한국어 음성이 기본으로 없는 PC가 많습니다. 재생이 안 되면 **설정 → 시간 및 언어 → 음성**에서 한국어 음성팩을 추가로 설치해주세요. |
| **Linux** | `espeak-ng`(또는 `espeak`) | 없다면 `sudo apt install espeak-ng`로 설치. 음질은 다소 기계적입니다. |

앱 실행 시 자동으로 설치된 한국어 음성 목록을 찾아 상단 드롭다운에 채워주고,
'유나(Yuna)' 계열 음성이 있으면 자동으로 우선 선택합니다(원본의 `loadVoices()`와 동일한 동작).

## 화면 구성 (원본과 동일)
- 장 선택 / 음성 선택
- 언어 모드: 한글+영어 / 한글만 / 영어만
- 구간(From~To) 설정
- 진행바, 절 카드(한글·영어 본문, 참조)
- 반복 횟수(±), 배속 슬라이더(0.50×~1.20×)
- 이전/재생·일시정지/다음 버튼
- 절 이동 슬라이더

## 참고 / 한계
- 이 개발 환경은 화면(디스플레이)이 없는 리눅스 샌드박스라서 실제 GUI 렌더링과 음성 출력은
  직접 확인하지 못했습니다. 컴파일과 클래스 로딩까지는 정상 확인했습니다(`java app.Main` 실행 시
  "HeadlessException"만 발생 — 디스플레이가 없어서 나는 정상적인 오류입니다).
- Windows에서 한국어 SAPI 음성이 설치되어 있지 않으면 한국어 문장이 소리 없이(또는 영어로) 재생될 수 있습니다.
